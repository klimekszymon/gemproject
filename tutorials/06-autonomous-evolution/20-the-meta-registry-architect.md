# Task 20: The Meta-Registry Architect (Capstone Project)

Congratulations! You have reached the final stage of the Skill Platform Architect curriculum. In this task, you will build the ultimate "Meta-Skill" – a tool that can autonomously architect and scaffold an entire **Domain Track** for the registry based on a single user instruction.

---

## Part 1: Educational - Platform Orchestration

Throughout this curriculum, you have been building individual skills. Now, you will shift from being a developer to being a **Platform Architect**. A Meta-Registry Architect skill doesn't just build a tool; it builds a **Framework**:

1.  **Requirement Synthesis**: Understanding the user's high-level goal (e.g., "Build a track for Python Data Science").
2.  **Structural Blueprinting**: Defining the folders, files, and tasks needed for a complete learning track.
3.  **Autonomous Scaffolding**: Using standard tools (like `mkdir` and `write_file`) to create the physical infrastructure of the new track.
4.  **Content Population**: Generating the educational content and initial task files for each step in the track.

---

## Part 2: The Practical - The `meta-architect` Skill

The `meta-architect` skill will be the "Admin" of your platform. It will be responsible for expanding the curriculum autonomously.

### 1. The Scenario
The user says: *"I want a new domain track for 'Game Development with Three.js' that covers scene setup, geometry, and basic animation."*

The `meta-architect` will then:
- Create `tutorials/07-game-dev/`.
- Create `21-scene-setup.md`, `22-geometries.md`, and `23-animations.md`.
- Populate these files with educational content and tasks.

## Part 3: Ready Scripts (The "Architect" Pattern)

### 1. `meta-architect/scripts/scaffold-track.js`
This script acts as the "Engine" that builds the directory structure and populates initial files.

```javascript
const fs = require('fs');
const path = require('path');

const trackName = process.argv[2];
const trackDir = path.join('tutorials', trackName.toLowerCase().replace(/\s+/g, '-'));

if (!fs.existsSync(trackDir)) {
  fs.mkdirSync(trackDir, { recursive: true });
  console.log(`📁 Created track directory: ${trackDir}`);
}

// Initial README for the track
const readmeContent = `# ${trackName} Domain Track\n\nWelcome to the ${trackName} track. This track was autonomously generated by the Meta-Registry Architect.`;
fs.writeFileSync(path.join(trackDir, 'README.md'), readmeContent);

console.log(`✅ Scaffolding complete for ${trackName}. Initial README.md created.`);
```
